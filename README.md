# System Zarządzania Treningami Fitness

## Opis tematu
Aplikacja konsolowa napisana w Pythonie, służąca do zarządzania planami treningowymi w klubie fitness. 
Umożliwia tworzenie różnych typów ćwiczeń (bieganie, pływanie, siłownia), przypisywanie ich do planów treningowych 
oraz zarządzanie użytkownikami (klubowiczami i trenerami). System oblicza spalone kalorie i śledzi postępy treningowe.

## Cele projektu
Głównym celem było praktyczne zastosowanie:
- czterech zasad OOP (enkapsulacja, dziedziczenie, polimorfizm, abstrakcja)
- relacji między klasami (kompozycja, agregacja, kolekcje, identyfikatory)
- konstruktorów, właściwości i metod

## Uruchomienie

**Plik startowy** – `main.py`  
- To jedyny plik, który należy uruchomić  
- Pozostałe pliki w folderze `models/` są automatycznie importowane

**Uruchomienie w terminalu** (z poziomu głównego folderu projektu):
```bash
python main.py
